## PostCraft Email Template Builder

An open-source Notion-style Email Template Builder with AI-powered autocompletions.

### Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open <http://localhost:5173> with your browser to see the result.

You can start editing by modifying `src/App.tsx`. The page auto-updates as you edit the file.

### Build

To build the project for production:

```bash
pnpm build
```

To preview the production build:

```bash
pnpm preview
```
